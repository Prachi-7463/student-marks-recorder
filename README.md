1. Student Marks Recorder
   
A menu-driven Python program built as part of my Python learning journey.

3. Description

Student Marks Recorder allows users to add student names and marks, save the records in a text file, and view all previously stored records.

The project uses file handling so that student records remain saved even after the program is closed.

3. Features

- Add a student's name and marks
- Store student records in a text file
- View all saved student records
- Menu-driven command-line interface
- Exit the program safely

4. Technologies Used

- Python
- Text File Handling

5. Concepts Practiced

- `while` loops
- Conditional statements
- User input handling
- File handling
- Writing to files using append mode
- Reading data from files
- Menu-driven program logic

6. How It Works

- Choose **Add Students** to enter a student's name and marks.
- The program saves the record in a text file.
- Choose **View All Students** to display every saved record.
- Choose **Exit** to close the program.

7. What I Learned

- How to create a menu-driven program
- How to store data permanently using a text file
- The difference between append mode (`"a"`) and read mode (`"r"`)
- How to read and display stored data
- How loops and conditions work together in a real project
- "w" mode works as write and delete and "a" mode works as adding one after the other
- For menu - driven programs, opening a file inside the option where we need it and close it there only
